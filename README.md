# Ambiente de Estudos - Django Web Framework (DRF)

Este repositório será usado como base prática para acompanhar o curso:

Curso de Django Web Framework com Python, HTML e CSS (Udemy)

Objetivo: construir aplicações web progressivamente, saindo do básico até recursos mais completos com Django, organização de projeto, templates, formulários, autenticação e APIs.

## Objetivo do Projeto

Criar um ambiente de aprendizado estruturado para desenvolver projetos reais com Django, aplicando boas práticas de desenvolvimento backend e frontend.

Ao final, a proposta é ter:

- Projetos Django funcionais e organizados.
- Entendimento de fluxo MVC/MTV no Django.
- Conhecimento de integração entre backend (Python/Django) e frontend (HTML/CSS).
- Base para evoluir para APIs com Django REST Framework (DRF).

## O que será desenvolvido

Durante o curso e neste repositório, serão desenvolvidos (de forma incremental):

- Configuração inicial de ambiente e projeto Django.
- Estrutura de apps, rotas e views.
- Templates HTML e estilização com CSS.
- Modelos (models), migrations e persistência em banco de dados.
- Formulários e validações.
- Área administrativa com Django Admin.
- Sistema de autenticação (login, logout, cadastro e permissões).
- CRUD completo de entidades do sistema.
- Organização de arquivos estáticos e media.
- Deploy e ajustes de produção (conforme avanço do curso).
- Introdução/expansão para APIs com DRF.

## Tecnologias Principais

- Python
- Django
- Django REST Framework (DRF)
- HTML5
- CSS3
- Banco de dados relacional (SQLite no início)

## Plano de evolução (roadmap)

- Etapa 1: Preparar ambiente, criar projeto e app principal.
- Etapa 2: Implementar páginas com templates e rotas.
- Etapa 3: Modelar dados e realizar CRUD.
- Etapa 4: Implementar autenticação e autorização.
